Todolist¶

This collection lists the to-do tasks for everyone.

Schema¶

The following lists key names mapped to its type and meaning for each entry.

_id

string, Unique project identifier., required

todos

list, a list of the todo tasks, required

type

dict, optional

description

string, the description of the to-do task, optional

due_date

string, the due date, optional

begin_date

string, the begin date, optional

end_date

string, the end date, optional

duration

float, the size of the task/ the estimated duration it will take to finish the task. Unit: miniutes., optional

importance

integer, the importance, from 0 to 2, optional

status

string, the status: proposed/started/finished, optional

notes

list, additional notes for this task, optional

type

string, optional

mark

integer, use this mark to update the task, optional

YAML Example¶

sbillinge:
  todos:
    - begin_date: '2020-06-15'
      description: read paper
      due_date: '2020-07-19'
      duration: 60.0
      importance: 2
      mark: 0
      status: started
    - begin_date: '2020-06-22'
      description: prepare the presentation
      due_date: '2020-07-29'
      duration: 30.0
      importance: 0
      mark: 1
      notes:
        - about 10 minutes
        - don't forget to upload to the website
      status: started
